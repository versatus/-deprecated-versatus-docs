# versatus-javascript

### required dependencies 
* node
* yarn
* javy


### install
```bash
yarn install @versatus/versatus-javascript
```

### initialize basic example
```bash
vsjs init
```

### initialize erc-20 example
```bash
vsjs init erc-20
```

### build contract
```bash
vsjs build example-contract.js
```

### test contract 
```bash
vsjs test inputs/sample-contract-input.json
```
